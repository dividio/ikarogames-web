PROJECT FOOTBALL

Introducción

Project Football is a football (soccer) management simulation game which allows you to manage the football club of your dreams. It allows you to manage financies, adjust tactics, buy players and watch matches in 3D.

This is a free software project under the GPL 2 license for source code and Creative Commons BY-SA 3.0 for the data files. If you want more information about Project Football or you want to colaborate, visit Ikaro Games' blog or our forum.
