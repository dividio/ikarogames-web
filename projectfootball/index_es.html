Introducción

Project Football es un completo y realista manager de fútbol con el que podrás convertirte en el gestor del club de tus sueños: realizar entrenamientos, controlar las finanzas, gestionar tus própias tácticas, fichar a los mejores jugadores, simulación de partidos en 3D, etc.

Este proyecto es software libre, se distribuye bajo las licencias GPL 2 para el código fuente y Creative Commons BY-SA 3.0 para los ficheros de datos. Si quieres más información sobre Project Football o deseas colaborar, visita el blog de Ikaro Games o nuestro foro.
